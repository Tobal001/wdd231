TobalCodes

I've chosen the domain name TobalCodes because it reflects both a personal and thematic connection. "Tobal" comes from my family nickname, derived from the latter half of my name, Cristobal, giving the site a personal touch. "Codes" narrows the focus of the site, clearly indicating that it revolves around programming. Together, the name represents my personal journey in programming and learning.

Proposed Domain Name: TobalCodes.ca Available

Purpose

The site serves a dual purpose: a place to document and organize the things I’m learning on my programming journey and a platform to share those insights with others who are also learning.

Components

  1. Programming Lingo (Glossary)

    As a first-year software development student, I’ve been bombarded with terminology that hasn’t quite stuck yet—especially since some concepts seem to have multiple names! This section will:

    • Provide clear definitions of terms.
    • Offer a searchable resource to quickly find information.
    • Help make sense of programming jargon.

  2. Fundamental Programming Concepts

    This section will focus on core concepts, featuring:

    • Simple code examples to illustrate each concept.
    • A feature to toggle between different programming languages (Python, C#, JavaScript), highlighting both subtle differences and similarities in syntax.
    • Real-world examples or projects demonstrating these concepts to keep them grounded and practical.
